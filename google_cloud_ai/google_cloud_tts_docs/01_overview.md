# Cloud Text-to-Speech Documentation

Cloud Text-to-Speech converts text or Speech Synthesis Markup Language (SSML) input into audio data of natural human speech.

## Key Service Features

The platform offers multiple models including Gemini-TTS, Chirp 3 HD voices, and instant custom voice capabilities. Users can synthesize speech through client libraries, command-line interfaces, or REST/RPC APIs.
